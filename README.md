# 宣传物料敏感词检索网页

这是一个用 `HTML + CSS + JavaScript` 原生写的本地小工具。  
不需要装框架，也不需要启动复杂环境，适合直接在电脑上打开使用。

## 怎么打开网页
最简单的方法：

1. 打开这个文件夹：`/Users/Zhuanz1/Desktop/vibecoding学习/敏感词检索器`
2. 找到 [index.html](/Users/Zhuanz1/Desktop/vibecoding学习/敏感词检索器/index.html)
3. 直接双击 `index.html`
4. 浏览器打开后就可以开始使用

1. 用 VS Code 打开这个文件夹
2. 安装 `Live Server`
3. 右键 `index.html`
4. 点 `Open with Live Server`

## 怎么使用
1. 打开网页后，左侧大输入框里可以直接粘贴文案
2. 点击“开始检测”
3. 页面会自动：
   - 在原文里高亮敏感词
   - 在右侧显示命中词、分类、风险等级、替换建议、备注
   - 计算整体风险等级
4. 如果你要补充新的词库，可以先点击“下载 CSV 模板”
5. 按模板填写后，再点击“上传 CSV 补充词库”
6. 每次检测记录会自动保存在浏览器本地，刷新页面后还在
7. 新上传的词库会叠加到现有词库上，不会把原词库删掉

## 词库格式要求
上传的补充 CSV 词库，请尽量保持下面这 5 列，顺序也保持一致：

```text
word,category,risk_level,replace,note
```

例如：

```text
提分,效果承诺,高,学习进步/能力提升/素养提升,涉及成绩承诺
包过,承诺类,高,提供学习支持,
轻松,夸大宣传,中,更有方法,
```

如果你不想自己从零建表，直接下载页面里的“CSV 模板”再填写会更稳。
补充词库上传后会和现有词库一起生效；如果出现同名敏感词，会优先使用你刚上传的那条。

## 如果打不开网页，怎么排查
你可以按下面顺序排查：

1. 先确认是不是打开了 `index.html`
   - 不要点错成 `README.md` 或 `敏感词词库.csv`

2. 如果页面样式乱了
   - 看看 `style.css` 和 `index.html` 是不是还在同一个文件夹里

3. 如果页面能打开，但默认词库提示读取失败
   - 这是有些浏览器对本地文件读取比较严格
   - 这个项目已经做了兜底处理，会自动切换到内置默认词库，正常检测通常不受影响

4. 如果你想让默认 CSV 一定按文件读取
   - 建议用 VS Code 的 `Live Server` 打开
   - 或者用本地静态服务器打开这个文件夹

5. 如果上传 CSV 后没反应
   - 检查文件是不是 `.csv`
   - 检查表头是不是 `word,category,risk_level,replace,note`
   - 检查每一行是不是按英文逗号分隔
